# Kubernetes Certification Killercoda Scenarios

This repository contains hands-on Killercoda scenarios for Kubernetes
certification preparation.

## CKA 2026

The `CKA_2026` scenarios are based on exact CKA 2026 exam questions, as per
my knowledge. They are designed to help practice the same style of Kubernetes
administration tasks expected in the exam environment.

## CKS 2026

The `CKS_2026` scenarios are based on CKS-style security questions and are
organized question by question.
